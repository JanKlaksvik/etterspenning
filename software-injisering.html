Injisering – borehull

Beregningen estimerer teoretisk volum for injisering basert på borehulls- og foringsrørdata. Prosjektleder kan sende ankerlisten til feltpersonell for utførelse og få data tilbake til godkjenning.

Prosjektstyring

Input

Prosjekt + innhold i hull + borehull
Velg eksisterende prosjekt, eller fyll inn nytt prosjekt og trykk «Lagre» for å lagre og åpne injisering felt.

Hva står i hullet?

Volumet som fortrenges av stang/spenntau trekkes fra borehullsvolum langs Fri + Forankring.

Borehull

Kontroll per anker: Foringsrør + (Borehull + overboring) skal være lengre enn Fri + Forankring.

Borehull (lengder i meter)

Anker nr. Foringsrør Borehull + overboring Fri Forankring Handling / kontroll
Volumberegningen bruker total borehullslengde definert som: Foringsrør + (Borehull + overboring).

Resultater

Volum + Ready Mix
Legg inn borehull, så beregnes volum automatisk.

Svinn / overforbruk

Merk: Beregningen inkluderer ikke masse til prøver.

Feltflyt

Prosjektet er ikke sendt til felt enda.
Feltpersonell kan registrere per anker: beregnet/brukt sement (sekker), start/stopp injisering, beregnet tid, trykkstabilitet (5-10 bar) og grout-temperatur. I tillegg registreres prosjektets oppstartskontroller (3-timers test, 1-liters vekt, siv-test og spread test).
